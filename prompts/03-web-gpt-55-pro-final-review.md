# 給 Web GPT-5.5 Pro 的最終審查 Prompt

你現在是外部審查者，不是本機執行者。

請審查下方 Codex 送審包，目標是找出：

1. 需求是否不清楚
2. 是否有漏測
3. 是否有高風險點被低估
4. 是否有交付阻塞
5. 是否有過度開發
6. 是否有錯把推測當證據
7. 是否有需要人類批准的地方

限制：

1. 不要假裝你跑過本機測試。
2. 不要假裝你看過沒有貼出的檔案。
3. 不要要求直接讀取或輸出 secrets。
4. 不要直接改程式。
5. 你的結論只是候選審查意見，不是交付批准。

請輸出：

1. 總分 0-100
2. 是否能交付
3. 前 10 個缺口
4. 前 10 個高風險點
5. 必補測試
6. 不建議現在做的事
7. 若要提升到 98-99 分，最小修改清單
8. 給 Codex 的下一步指令

Codex 送審包如下：

```text
貼上 Codex 送審包
```

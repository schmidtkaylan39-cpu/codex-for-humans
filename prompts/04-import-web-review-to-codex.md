# 把 Web GPT 審查結果交回 Codex

使用 `$nontechnical-codex-project-controller`

下面是 Web GPT 的候選審查意見。請不要直接照單全收。

請先分類：

1. 接受，這輪執行
2. 接受，但排到下一輪
3. 不採用，並說明原因
4. 需要人類批准
5. 需要更多本機證據
6. 只是模型成本建議，先做 Model Cost Routing Check 再決定

規則：

1. Web GPT 不是本機證據。
2. 最終以本機檔案、測試、執行結果、截圖、報告為準。
3. 不要默默擴大任務範圍。
4. 高風險操作要先等我明確批准。
5. 請產出最小可執行收尾任務。
6. 不要因為 Web GPT 建議便宜模型，就把高風險證據或最終交付判斷交給便宜模型。

# 公開前檢查表

把 repo 設成公開前，先做這份檢查。

## 必做

- [ ] 確認 repo 內沒有 `.env`
- [ ] 確認 repo 內沒有 API key、password、token、cookie
- [ ] 確認 repo 內沒有私人聊天紀錄
- [ ] 確認 repo 內沒有 Discord、交易所、雲端、付款服務憑證
- [ ] 確認 repo 內沒有真實客戶資料
- [ ] 確認 repo 內沒有私人專案 logs
- [ ] 確認 README 說明清楚
- [ ] 確認兩個 Skill 都在 `skills/`
- [ ] 確認 prompt 都在 `prompts/`
- [ ] 確認 `.gitignore` 已建立

## 建議先私有

建議流程：

1. 先建立 private GitHub repo。
2. 上傳後再用 GitHub 網頁檢查一次。
3. 確定沒有敏感內容後，再切成 public。

## 一句話原則

```text
只公開可重用流程，不公開你的私人執行紀錄。
```

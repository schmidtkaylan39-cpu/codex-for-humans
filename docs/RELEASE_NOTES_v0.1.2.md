# Codex for Humans v0.1.2

This release is the first public-ready package for Codex for Humans.

## Included

- Two Codex Skills:
  - `nontechnical-codex-project-controller`
  - `nontechnical-project-readiness-auditor`
- Beginner prompts for 0-1 project work, 70-100 readiness audits, Web GPT review, and Web-to-Codex import.
- Project templates for `AGENTS.md`, project rules, decision logs, evidence ledgers, risk registers, release checks, and Web review packets.
- Visual onboarding:
  - Mermaid workflow map
  - SVG install flow
  - animated GIF install demo
- One-click install scripts:
  - `scripts/install.ps1`
  - `scripts/install.sh`
- Traditional Chinese and English clinic booking system walkthroughs.
- Security and public-readiness guardrails.
- Lightweight GitHub Actions repo checks.

## Safety Reminder

This is an unofficial workflow kit. It does not guarantee that generated software is correct, secure, legal, deployable, or safe for real money.

Use sandbox, fake data, test accounts, dry runs, or local simulation first. Do not paste secrets into chat. High-risk actions require explicit human approval and rollback planning.
